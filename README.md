A simple mod that allows Waterleaf to bloom in water and Fireblossom to bloom around lava instead of only at sunset, featuring a customizable configuration.

The toggles for Require Submergence only has to do with Whether the mod checks a 3x4 area or only a 1x2 area, when it is off it checks a 3x4 area around the plant, that reaches to one block left, one block right, one block up, and two blocks down. 

When Submergence is toggled on it instead only checks an area of 1x2, the plant and the block below the plant which allows the player to submerge only the planter instead of the plant itself.

The toggles for boost applies the same boost Waterleafs recieve in rain but constantly when submerged, it only ever will track the 1x2 area not 3x4 regardless of whether submergence is required.

If submergence requirements are toggled on for Fireblossom it should still naturally grow outside of lava, toggling them on only changes the area of which the mod registers submergence around the plant.
